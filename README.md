\# 🏆 Campeonato de Futebol



Projeto desenvolvido em Java como prática dos conceitos de Programação Orientada a Objetos.



O sistema permite cadastrar equipes, registrar partidas e gerar automaticamente a classificação final de um campeonato de futebol.



\## Funcionalidades



\- Cadastro de times participantes;

\- Registro de partidas;

\- Processamento automático dos resultados;

\- Exibição do campeão;

\- Exibição dos resultados das partidas;

\- Classificação final ordenada por:

&#x20; - Pontos;

&#x20; - Saldo de gols;

&#x20; - Gols marcados.



\## Conceitos praticados



\- Programação Orientada a Objetos (POO)

\- Encapsulamento

\- Classes e Objetos

\- Collections (`List`)

\- Interface `Comparator`

\- Sobrescrita de `toString()`

\- Organização em pacotes

\- Refatoração de código
